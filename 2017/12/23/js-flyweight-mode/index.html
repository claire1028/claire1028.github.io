JavaScript享元模式

享元模式：

剥离出对象的外部状态和内部状态，将内部状态放在类中，外部状态可以通过同一个对象共享，即用相对较少的共享对象取代大量对象，常用在创建很多类似的对象而导致内存过高的场景中。
